# 프로그래머스 연습문제 240826

문제 : [가장 가까운 같은 글자](https://school.programmers.co.kr/learn/courses/30/lessons/142086)  
난이도 : Lv.1

## 풀이

``` javascript
   function solution(s) {
    
    const lastSeen = {};
    const result = [];    

    for (let i = 0; i < s.length; i++) {
        const char = s[i];

        if (char in lastSeen) {

            const lastPosition = lastSeen[char];
            result.push(i - lastPosition); 
        } else {
            result.push(-1);
        }

        lastSeen[char] = i;
    }

    return result;
    
}
```
